
  # Crear presentación desde sketch

  This is a code bundle for Crear presentación desde sketch. The original project is available at https://www.figma.com/design/5TcgqJBs0iRlgYakqVAiHL/Crear-presentaci%C3%B3n-desde-sketch.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Exporting to PDF

  Install the Playwright browser once:

  ```bash
  npm run playwright:install
  ```

  Export the 8-slide presentation to `../docs/Distributed_File_System_Presentation.pdf`:

  ```bash
  npm run export:pdf
  ```
